# Security Policy

## Supported Versions

Only the latest release published on the
[Gradle Plugin Portal](https://plugins.gradle.org/plugin/io.github.ahmetsirim.kot) receives
security fixes; there are no maintenance branches for older versions.

## Reporting a Vulnerability

Please use
[GitHub's private vulnerability reporting](https://github.com/AhmetSIRIM/kot/security/advisories/new)
for suspected vulnerabilities; it opens a private channel to the maintainer, and nothing becomes
public before a fix is available. Please do not report suspected vulnerabilities through public
issues.

You can expect an initial response within a week. An accepted report is fixed in a new release
first; the advisory is published after the fix is available.
